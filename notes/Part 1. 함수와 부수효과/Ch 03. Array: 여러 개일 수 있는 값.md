# Clip 1. 돌아보기 > 함수와 함수 합성

## 명령형에서 함수형 추상화로

명령형 틀에 박힌 코드

```javascript
function sum1To100() {
  let sum = 0;
  for (let i = 1; i <= 100; i++) {
    sum += i;
  }
  return sum;
}
```

함수형 추상화

```
range(1, 100)

입력(매개변수)

loop
const plus = (a, b) => a + b

출력(반환)

5050
loop(plus, 0, range(1, 100))
```
